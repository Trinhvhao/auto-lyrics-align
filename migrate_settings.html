🔄 Migrate Settings - Đổi tên file

📋 Bước 1: Chọn bài nhạc cần migrate

Click vào bài nhạc bạn muốn đổi tên

✏️ Bước 2: Nhập tên file mới

Nhập tên file mới (bao gồm .mp3)

💡 Tip: Tên file phải giống CHÍNH XÁC với file bạn sẽ upload

🎯 Bước 3: Migrate

ℹ️ Hướng dẫn

  1. Chọn bài nhạc từ danh sách
  2. Nhập tên file mới (ví dụ: music1.mp3)
  3. Bấm "Migrate Settings"
  4. Settings sẽ được copy sang tên file mới
  5. Bạn có thể xóa bài cũ nếu muốn
  6. Reload trang karaoke và chọn file với tên mới
